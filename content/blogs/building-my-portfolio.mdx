Welcome to my first blog post! In this article, I'll walk you through the process of building my personal developer portfolio.

## Why Next.js?

I chose **Next.js** for its powerful features like Server Components, static site generation, and seamless routing. The App Router makes it incredibly easy to build layouts and manage data fetching.

### Key Features Used

1. **App Router**: For intuitive file-system based routing.
2. **Server Components**: To keep the bundle size small.
3. **MDX**: For writing these blog posts!

## Styling with Tailwind CSS

Tailwind CSS has completely changed how I write styles. Instead of switching between CSS files and React components, I can style everything directly in my markup.

```tsx
// Example of a styled button
export function Button() {
  return (
    <button className="rounded-lg bg-zinc-900 px-4 py-2 text-white hover:bg-zinc-800 dark:bg-zinc-100 dark:text-zinc-900 dark:hover:bg-zinc-200">
      Click Me
    </button>
  );
}
```

## Adding Animations

To make the site feel alive, I used **Framer Motion**. It's perfect for scroll animations and page transitions.
